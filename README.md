# Texas-Hold-em

## 简介

A Texas Hold'em WebGame based on Socket.io, Node.js and Express.

## 项目结构

```
Texas-Hold-em                   # 项目主文件
├── game_records/               # 保存历史记录
├── node_moudules/              # Node.js 依赖包
├── src/                        # 项目源码
│   ├── client/                 # 前端代码
│   │   ├── css/                # 样式文件
│   │   │   ├── index.css       # 主样式文件
│   │   │   └── materialize.css # Materialize 框架样式
│   │   ├── img/                # 图片资源
│   │   ├── js/                 # JavaScript 文件库
│   │   ├── index.html          # 主页面
│   │   └── main.js             # 前端主要逻辑
│   └── server/                 # 后端代码
│       ├── app.js              # 服务器主文件
│       ├── database.json       # 数据库文件
│       └── game.js             # 后端主要代码
├── package.json                # 项目配置文件
├── README.md                   # 项目说明文档
└── start.bat                   # Windows 快速启动脚本
```
